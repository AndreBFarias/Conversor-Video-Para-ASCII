Itens Pendentes para Proxima Release

Fallbacks Hardcoded (baixa prioridade)
Restam ~35 fallbacks hardcoded em converters que usam valores corretos mas nao importam de `get_default()`. Os valores estao corretos (alinhados com defaults.py), mas por consistencia seria ideal migrar todos para `get_default()`.

Arquivos afetados: converter.py, mp4_converter.py, gpu_converter.py, gif_converter.py, html_converter.py, png_converter.py, realtime_ascii.py, gtk_fullscreen_player.py, preview_actions.py

print() Residuais no realtime_ascii.py
29 print() fora de __main__ em `src/core/realtime_ascii.py`. Modulo usa print extensivamente para debug de real-time. Migrar para logger quando o modulo for refatorado.

Sprint 41: Conflitos de Features (planejada)
Documentacao completa em `docs/SPRINT_41_CONFLITOS_FEATURES.md`. Implementacao da funcao `_validate_features()` e verificacao pre-conversao.
