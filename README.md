### Descrição
Um ritual de magia digital que transmuda vídeos em animações ASCII coloridas, banindo fundos indesejados e evocando silhuetas em caracteres que dançam no terminal ou em janelas flutuantes.

### Instalação e Uso

1. **Clone o Repositório:**
   ```bash
   git clone https://github.com/AndreBFarias/Conversor-Video-Para-ASCII.git
   cd Conversor-Video-Para-ASCII
   ```

2. **Execute o Ritual de Consagração:**
   ```bash
   chmod +x install.sh
   ./install.sh
   ```

3. **Inicie o Grimório:**
   ```bash
   python3 main.py
   ```

### Estrutura do Projeto
- `src/`: Lógica principal e GUI.
- `assets/`: Artefatos visuais.
- `logs/`: Registros de execução.
- `data_input/`: Coloque seus vídeos aqui.
- `data_output/`: Recolha suas artes ASCII aqui.
