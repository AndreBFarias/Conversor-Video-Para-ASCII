# Release v2.0.0 - Multimodalidade

> **TL;DR:** Release que trouxe suporte a voz (Whisper + TTS), visao (Gemini Vision) e sistema de threading paralelo.

## Contexto

A versao 2.0 foi uma refatoracao major que transformou Luna de um chatbot de texto em uma assistente multimodal completa. Introduziu capacidades de voz bidirecional e processamento de imagens.

## Data de Lancamento

**2024-11-30**

## Highlights

1. Suporte completo a voz (STT + TTS)
2. Processamento de imagens via Gemini Vision
3. Threading paralelo para operacoes assincronas
4. Integracao com Google Gemini

## Changelog Detalhado

### Adicionado

- **Speech-to-Text**: Faster-Whisper com modelos configuráveis
- **Text-to-Speech**: Suporte a ElevenLabs e Coqui TTS
- **Vision**: Processamento de imagens via webcam
- **Threading**: Sistema paralelo para I/O
- `voice_system.py`: Modulo de voz completo
- `visao.py`: Modulo de visao computacional
- `threading_manager.py`: Gerenciador de threads
- Deteccao de silencio via WebRTC VAD
- Cache de respostas para otimizacao

### Alterado

- Integracao com Google Gemini API
- Interface TUI redesenhada para multimodalidade
- Sistema de prompts adaptado para visao
- Config expandido com parametros de voz e video

### Corrigido

- Latencia em processamento de audio
- Qualidade de transcricao em ambientes ruidosos
- Sincronizacao de audio/texto

## Arquitetura v2.0

```
Luna/
├── main.py                  # Orquestrador
├── config.py               # Configuracoes (.env)
├── src/
│   ├── luna/
│   │   ├── consciencia.py      # Gemini LLM
│   │   ├── boca.py             # TTS
│   │   ├── voice_system.py     # STT + VAD
│   │   ├── visao.py            # Webcam + Vision
│   │   └── threading_manager.py # Threads
│   └── ui/
│       ├── widgets.py          # Componentes
│       └── screens.py          # Telas
```

## Mudancas de Breaking

1. `config.ini` substituido por `.env`
2. Provider padrao: Gemini
3. Estrutura de diretorios reorganizada
4. Requirements separados (core + tts)
