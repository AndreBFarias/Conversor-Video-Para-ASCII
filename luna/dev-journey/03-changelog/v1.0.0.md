# Release v1.0.0 - Primeiro Release

> **TL;DR:** Release inicial com interface TUI basica, integracao com API LLM e personalidade gotica.

## Contexto

Este foi o primeiro release publico do projeto Luna. A versao 1.0 estabeleceu a base da arquitetura, a interface TUI com Textual e a integracao com API LLM.

## Data de Lancamento

**2024-10-15**

## Highlights

1. Interface TUI funcional com tema dark
2. Integracao com LLM
3. Personalidade gotica definida
4. Sistema de configuracao via config.ini

## Changelog Detalhado

### Adicionado

- Interface TUI basica com Textual framework
- Integracao com API LLM
- Sistema de input/output de texto
- Personalidade gotica inicial ("Luna")
- Configuracao via `config.ini`
- Logging basico em `logs/`
- README com instrucoes de instalacao
- Licenca GPLv3
- Estrutura de diretorios padrao

### Caracteristicas Iniciais

- Processamento de texto via LLM
- Respostas com personalidade sarcastica
- Tema dark basico (Dracula)
- Comandos de controle (exit, clear, help)

## Arquivos Principais

```
Luna/
├── main.py              # Entry point inicial
├── config.ini           # Configuracoes
├── requirements.txt     # Dependencias
├── LICENSE              # GPLv3
├── README.md            # Documentacao
└── src/
    ├── luna/

    └── ui/
        └── interface.py    # TUI basica
```

## Limitacoes Conhecidas

- Apenas texto (sem voz ou visao)
- Sem memoria entre sessoes
- Sem threading (bloqueante)
- Tema visual basico

## Metricas

- Linhas de codigo: ~1.500
- Arquivos Python: 8
- Dependencias: 5
- Tempo de resposta: ~3s
